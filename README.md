# hbn
**hbn** - http benchmark tool written on golang

---

Fast, simple and powerfull!!!

---

## Installation

In UNIX based:
```shell
$ make
```

In windows:
```shell
$ go build -v ./src/hbn.go
```

## Usage

```shell
$ hbn -url <host to attack> -c <count of threads> -d <duration of testing>
```

## Help

```shell
$ hbn -h
```

or
```shell
$ hbn --help
```

or
```shell
$ hbn -help
```
